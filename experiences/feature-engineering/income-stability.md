# 收入稳定性特征衍生方案

## 来源
- 项目：消费贷风控项目
- 日期：2025-04-30
- 场景：预测个人消费贷违约风险

## 经验内容

### 核心思路
对收入类变量，不要只看绝对值，要计算"波动率 + 趋势 + 突变"三个维度：

1. **波动率**：近6个月收入的标准差/均值（CV系数）
   - CV > 0.5 → 收入不稳定，违约概率显著上升
   - 需要处理零值收入（失业月份），建议用 (σ + 1) / (μ + 1) 避免除零

2. **趋势**：近6个月收入的线性回归斜率
   - 斜率为负 → 收入下降趋势，风险信号
   - 建议归一化：slope / mean_income

3. **突变**：最大单月降幅 / 均值
   - 突降 > 50% → 强风险信号
   - 需排除正常的季节性波动（如年终奖月份）

### 组合特征
- income_volatility x debt_ratio → 高负债+高波动 = 极高风险
- income_trend x delinquency_history → 收入下降+有过逾期 = 风险叠加

## 适用条件
- 有月度收入流水数据（至少6个月）
- 适用于个人信用贷款、消费贷场景
- 目标变量为是否违约/逾期

## 注意事项
- 小微企业主收入波动天然较大，CV阈值需要调整（可能 0.7 以上才算不稳定）
- 零工经济从业者（外卖、网约车）收入波动大但不等于高风险，需要结合行业调整
- 收入数据缺失率通常较高（30%+），需要先做缺失值处理策略
