# Chris Risk Workbench 🏦

信贷风控多 Agent 工作台 — OpenCode 插件

覆盖 **数据分析、特征衍生、策略制定** 三条完整链路。9 个专业 Agent 协同作战。

## 快速开始

```bash
# 构建
npm install && npx tsc

# 在 opencode.json 中配置
{
  "plugin": [
    "file:///path/to/chris-risk-workbench/dist/index.js"
  ]
}
```

## 9 个 Agent
| Agent | 角色 | 模型 | 什么时候用 |
|-------|------|------|-----------|
| **Chris** | 主编 | glm-5.1 | 永远 — 和 Chris 对话就行 |
| **分析师** | 数据分析 | glm-5.1 | 数据探索、统计分析、IV/WOE 计算 |
| **质疑员** | 逻辑辩论 | MiniMax-M2.7 | 审查结论和策略的合理性 |
| **工程师** | 特征工程 | glm-5.1 | 特征衍生、代码生成 |
| **研究员** | 深度搜索 | glm-5.1 | 行业研究、竞品分析（串行：先搜后研） |
| **跨界顾问** | 创意联想 | MiniMax-M2.7 | 提供跨领域视角，辩论时 |
| **框架师** | 方法论推荐 | glm-5.1 | 推荐分析框架、搜索知识库时 |
| **进化师** | 系统改进 | glm-5.1 | 复盘、评估、知识整理、跨项目经验沉淀 |
| **视觉员** | 多模态 | glm-5v-turbo | 图片OCR、图表解读、证件识别 |

## 核心特性

- **辩论机制** — 质疑员 + 跨界顾问多角度辩论，避免盲区
- **24 个框架知识库** — 4 层分析方法论，按需查询
- **串行研究工作流** — parallel-deep-research 采集素材 → hv-analysis 横纵分析 → 万字报告
- **多项目知识管理** — 项目私有 `.chris-risk/` + 跨项目 `experiences/` 经验库
- **3 种工作模式** — 数据分析 / 特征衍生 / 策略制定, 可自动识别
- **灵活配置** — 每个Agent模型/温度/工具都可覆盖

## 知识架构

```
框架目录（共享）                    项目目录（隔离）
chris-risk-workbench/              {your-project}/
├── frameworks/    ← 通用方法论    ├── .chris-risk/
├── experiences/   ← 跨项目经验    │   ├── knowledge.md  ← 项目私有知识
└── skills/        ← 通用技能      │   └── decisions.md  ← 决策日志
                                   └── data/
```

- **"整理一下"** → 进化师自动同步项目知识，优秀的经验提示沉淀到框架
- **"调研一下"** → 研究员串行搜索+分析，产出结构化报告
- **新项目启动** → 自动加载跨项目经验摘要到上下文
